Build the server zip / exe / apps:

Windows (must be a Python 2.6 binary):
   python setup_win2exe.py py2exe

Mac OS X (must be a 32bit Python binary - (10.6 has a 64bit by default):
   python setup_py2app.py py2app


build client:

palm-package bt_client



install client:

palm-install *.ipk
